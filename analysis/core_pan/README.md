The core/pan-genome analysis of Salmonella enterica shows a moderately conserved genome, with core genes making up ~53–56% of the total and accessory genes accounting for the remaining ~44–47%, indicating both shared essential functions and genomic diversity across strains. Results from Panaroo and PPanGGOLiN were pretty consistent, differing by only ~80 genes, which increases confidence in the identified core genome despite their different methodologies (graph-based correction vs statistical partitioning). This is provides a good foundation for identifying candidate receptor genes.
